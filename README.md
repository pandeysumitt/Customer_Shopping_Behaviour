# Customer_Shopping_Behaviour
# Overview
This project analyzes customer purchasing behavior for a retail business to uncover actionable insights that drive revenue growth, improve customer retention, and optimize marketing strategies.
Using an end-to-end analytics workflow, raw transactional data is transformed into meaningful insights and interactive dashboards for business decision-making.

# Business Problem
How can a retail company understand customer purchasing behavior to increase revenue, improve customer retention, and optimize marketing strategies?

# Solution Approach 
* Performed customer segmentation using RFM (Recency, Frequency, Monetary) analysis
* Analyzed purchase patterns and customer trends
* Identified high-value and at-risk customers
* Conducted cohort analysis to track retention
* Built interactive dashboards for business insights

# Key Impact
* Identified top customer segments contributing the most revenue
* Detected churn-risk customers for targeted retention strategies
* Improved marketing targeting using behavioral insights
* Highlighted opportunities to increase repeat purchases

# Tech Stack
Excel – Data cleaning & preprocessing
SQL – Data extraction, joins, aggregations
Python (Pandas, NumPy) – Data analysis & transformation
Power BI – Dashboard & visualization

# End-to-End Workflow
1. Data Collection
Collected transactional and customer data
2. Data Cleaning
Handled missing values and duplicates
Standardized formats and created new features
3. Data Querying (SQL)
Extracted relevant datasets using joins and aggregations
4. Data Analysis (Python)
* Performed exploratory data analysis (EDA)
* Conducted RFM segmentation
*Identified trends and anomalies
5. Data Visualization (Power BI)
* Built interactive dashboards with KPIs
* Enabled filtering by customer segments and time

# Key Insights
* A small percentage of customers generated the majority of revenue (Pareto principle)
* High-frequency customers showed strong retention and repeat purchase behavior
* Certain customer segments had high churn risk due to low engagement
* Seasonal trends significantly impacted purchasing patterns
